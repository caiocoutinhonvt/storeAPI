
# storeAPI  

Standard e-commerce API create with Moleculer Framework
https://moleculer.services/


This API contains: 

- CRUD of Store, Product. 
- User register and authentication
- Shopping cart for users. 
- Upload a picture/video of your own products

Used Tools:

- Node.js
- Moleculer Framework
- Typescript
- Docker
- TypeORM
- PostgreSQL
